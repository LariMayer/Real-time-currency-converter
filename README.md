Sobre o projeto

Esse projeto nasceu da minha vontade de treinar APIs, manipulação do DOM, lógica assíncrona e atualização de valores em tempo real, mas sem perder o meu estilo.

Então eu criei um **Conversor Vintage**, inspirado em elementos retrô, com cores quentinhas, letra de datilografia e aquela vibe nostálgica.

Ele converte automaticamente valores de **Real (BRL)** para:

**Dólar (USD)**
**Euro (EUR)**
**Libra Esterlina (GBP)**
**Bitcoin (BTC)**

As cotações são atualizadas em tempo real usando uma API de mercado.
